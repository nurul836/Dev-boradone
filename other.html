Question-3: What is event delegation in the DOM?

Event Delegation means adding an event listener to a parent element to capture events that occur in any child element within that parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event occurs in a child, that event goes up to its parent, grandparent, and so on up the document. This means that the event goes from the bottom to the top.

Question-5: How do you create, add, and remove elements using JavaScript?
